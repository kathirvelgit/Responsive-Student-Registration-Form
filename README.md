# 🎓 Responsive Student Registration Form

A modern, responsive **Student Registration Form** built using **HTML5, CSS3, and JavaScript**. This project demonstrates form handling, DOM manipulation, responsive web design, client-side validation, and dynamically displaying submitted student information in a table without refreshing the page.

---

# ✨ Features

✅ Responsive Student Registration Form

✅ HTML5 Form Validation

✅ Required Input Fields

✅ Dynamic Table Row Creation

✅ Prevents Page Refresh using JavaScript

✅ Form Automatically Resets After Submission

✅ Responsive Design using Flexbox

✅ Mobile-Friendly Layout

✅ Horizontal Scroll for Table on Small Devices

✅ Clean & Modern User Interface

✅ Organized Code Structure

---

# ⚙️ Functionalities

### ✅ Form Validation

- Required fields validation
- Email validation
- Pincode validation
- Gender selection validation

---

### ✅ Dynamic Table

After clicking the **Submit** button:

- Student details are added as a new row in the table.
- Previous records remain visible.
- The form is automatically cleared.
- The page does not refresh.
